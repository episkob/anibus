# Anibus — Сканер портов

Современное десктопное приложение для сканирования портов, созданное с использованием **Anibus Design System**, **JavaFX 21.0.5** и **Java 21**.

> **Версия:** 1.1.0 · **Автор:** Iaroslav Tsymbaliuk · **Должность:** Intern (2025–2026) @ r2u

---

## Возможности

### Сканирование
- Сканирование любого имени хоста или IP-адреса на открытые порты
- Настраиваемый диапазон портов (например, `1-1024`)
- Регулируемое количество потоков (10–500, по умолчанию 100)
- Измерение **задержки (latency)** для каждого порта (мс)
- Плавная отмена сканирования

### Извлечение информации
- Автоматическое **определение сервиса** для 50+ известных портов (HTTP, SSH, FTP, MySQL, PostgreSQL, Redis, MongoDB, Elasticsearch, Kubernetes API и др.)
- **Захват баннеров** — HTTP-порты получают расширенные заголовки через `HEAD`-запрос
- **Извлечение версии ПО** из баннеров (OpenSSH, Apache, nginx, ProFTPD, Postfix и др.)
- Определение протокола и **шифрования** (TLS, STARTTLS, HTTPS, SMTPS, LDAPS и др.)

### Интерфейс и рабочий процесс
- **Панель информации о хосте** — IP, имя хоста, время сканирования, количество проверенных и открытых портов, средняя задержка (обновляется в реальном времени)
- **Таблица результатов с 7 колонками** — Порт, Состояние, Сервис, Версия, Протокол, Задержка, Баннер
- **Экспорт в CSV или XML** со всеми колонками
- **Очистка** результатов одним нажатием
- **Копирование строки** или **всех результатов** через контекстное меню
- Автоматическое определение DNS при потере фокуса
- **Anibus Design System**: стеклянная навигация, градиентные кнопки, тонкие полосы прокрутки, скруглённые карточки, цветовое кодирование состояния

## Структура проекта

```
src/
└── main/
    ├── java/
    │   ├── module-info.java
    │   └── it/r2u/anibus/
    │       ├── AnibusApplication.java     # Точка входа JavaFX
    │       ├── AnibusController.java      # Контроллер UI — связывает все сервисы
    │       ├── PortScanResult.java        # Модель данных (7 полей)
    │       ├── PortScannerService.java    # Координатор: проверка задержки и парсинг диапазона
    │       ├── ScanTask.java              # Фоновая задача Task<Void> с коллбэками
    │       ├── BannerGrabber.java         # Захват баннеров (HTTP HEAD / сырое приветствие)
    │       ├── VersionExtractor.java      # Извлечение версии ПО из баннеров (regex)
    │       ├── PortRegistry.java          # Таблицы сервисов и протоколов/шифрования
    │       ├── ExportService.java         # Экспорт в CSV и XML с диалогом выбора формата
    │       ├── TableConfigurator.java     # Настройка колонок и ячеек таблицы
    │       ├── ClipboardService.java      # Утилиты копирования в буфер обмена
            └── AlertHelper.java           # Стилизованные диалоги Anibus Design System
    └── resources/
        └── it/r2u/anibus/
            ├── hello-view.fxml            # Макет интерфейса
            ├── anibus-style.css           # Таблица стилей Anibus Design System
            └── app.properties             # Версия приложения (фильтруется Maven)
```

## Сборка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/episkob/anibus.git
cd anibus

# Запуск через Maven
./mvnw javafx:run
```

На Windows:

```cmd
mvnw.cmd javafx:run
```

## Использование

1. Введите **целевой хост** (имя хоста или IP-адрес)
2. Введите **диапазон портов** в формате `start-end` (например, `1-1024`)
3. При необходимости настройте **количество потоков**
4. Нажмите **Start Scan**
5. Появится **панель информации о хосте** с живой статистикой
6. Результаты заполняют таблицу по мере обнаружения портов
7. Нажмите **Stop** для прерывания сканирования в любой момент
8. Используйте **Export** для сохранения в **CSV или XML**, или **Clear** для сброса
9. Кликните правой кнопкой мыши для копирования одной строки или всех результатов
